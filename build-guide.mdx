## **4. Build Types**

You can generate **Debug**, **Profile**, or **Release** builds depending on your needs.

---

### **A) Debug Build (Local Development)**

```bash
flutter build apk
```

Output:

```
build/app/outputs/flutter-apk/app-debug.apk
```

Debug builds include:

* Debug banner
* Logs enabled
* Fast iteration

---

### **B) Profile Build (Performance Testing)**

```bash
flutter build apk --profile
```

Useful for animation/performance benchmarking.

---

### **C) Release Build (For Internal Distribution)**

```bash
flutter build apk --release
```

Output:

```
build/app/outputs/apk/release/app-release.apk
```

Release builds are optimized and suitable for testers.

### **Java Version Issues**

### **Android SDK Missing**

* USB Debugging
* Install Unknown Apps
