## 4. Compliance Boundaries (Important)

FinTra is not a financial institution and does not:
- Provide loans
- Store bank credentials
- Enable transactions
- Handle KYC/AML flows

This avoids heavy regulatory burdens while building the core product.

## 5. Data Retention & Deletion Policy

### Retention
- Minimal personally identifiable data
- All financial entries owned by the user
- Logs and analytics stored only for debugging and performance

### Deletion
- Users can request manual deletion (automated in Phase 2)
- Offboarding = permanent removal from database
- No shadow retention
