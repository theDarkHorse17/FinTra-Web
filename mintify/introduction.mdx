# FinTra Documentation

FinTra is a behaviour-driven personal finance assistant for India’s 21–33 earning population.  
Unlike traditional budgeting apps, FinTra focuses on *behaviour correction*, not data entry.

This documentation includes:

- Product & UX overview
- Technical architecture
- Supabase schema
- API specs
- Developer onboarding
- Security
- Investor readiness
- Roadmap
