# **FinTra Changelog**
### *Transparent record of every update shipped to users*

This changelog tracks all major improvements, fixes, and versions released for FinTra. Each entry follows semantic versioning (MAJOR.MINOR.PATCH).

---

## **v1.0.0 — MVP Launch (July 2025)**
**Added**
- Manual expense entry
- Five default categories: Investments, Necessity, Health, Charity, Fun
- Savings entry & monthly overview
- Graph comparing current vs previous month's spending
- Category-wise spending breakdown
- Login system
- Core UI and dark theme

**Notes**
- First stable MVP built solo
- Internal testing with early friends & users

---

## **v1.0.1 — Stability & UI Polish (August 2025)**
**Improved**
- Faster expense entry
- UI spacing, padding, gradients
- Animation smoothness
- Overall app responsiveness

**Fixed**
- Crash on first login
- Graph rendering delay

---

## **v1.1.0 — Insights Upgrade (September 2025)**
**Added**
- Monthly summary ring charts (Spent vs Saved)
- Improved category graph visuals
- Performance optimization for home screen

**Improved**
- Data loading time

---

## **v1.2.0 — Goal System + Image Preview (October 2025)**
**Added**
- Goal creation feature
- Automatic image fetching + preview
- Expandable goal container

**Improved**
- UI transitions

---

## **v1.3.0 — Behaviour Engine (November 2025)**
**Added**
- Smart recurring reminders (3/day)
- Early habit insights

**Improved**
- Notification delivery reliability
